HTML

  • HTML attributes are defined in the beginning
  • HTML elements are case sensitive
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML is used to create the structure of a website
  • HTML is made up of elements which work together to provide content and the basic framework for all pages on the web
  • HTML stands for Hypertext Markup Language

CSS

  • CSS is used to give style to the text
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allows us to share a CSS rule to any element we choose by assigning the rule to a class attritube with a class selector
  • The period before card designates this is a class selector
  • The box-shadow CSS property adds shadow effects around an element frame, and you can set multiple effects separated by commas
  • When we create an external CSS file, we need to link it to the HTML file. Just because our files are in the same project repository doesn't mean they know about each other

Git

  • Git command must procede every git statement
  • Git is the most commonly used version control software
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • To access Git we use the command line
  • Git is great to track changes locally on your own computer, but to collaborate with others we use Github.

JavaScript

  • We use JS to handle logic and allow users to interact with a website
  • A variable is a named container that allows us to store date in our code
  • Control Flow is the order in which a computer executes code in a script
  • A function is simply a set of instructions that tells the computer how to perform a certain task
  • Functions do not automatically execute when the Javascript file is runned, you must call a function by name in order to execute it
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
  • In JS arrays are zero-indexed and sequential
  • Arrays are not limited to contain only a single type of data
  • An array is a single variable that is used to hold a group of data
  • A variable is like a container that allows us to store data on our code
  • In JS to declare a variable we use the var keyword
